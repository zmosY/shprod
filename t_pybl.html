В этом уроке мы разберем основные концепции программирования через призму Python и блочного программирования Blockly.

1. Присваивание (Assignment) (находится в разделе Переменные/Variables)

Операция присваивания позволяет сохранить значение в переменной. Это как положить предмет в названную коробку - теперь мы всегда знаем, где его найти. Например, можно сохранить число 5 в переменной x.

2. Вывод (Print) (находится в разделе Текст/Text)

Вывод - это способ показать результат работы программы пользователю. Это как написать что-то на доске - информация становится видимой для всех. Мы можем выводить текст, числа и результаты вычислений.

3. Циклы (Loops) (находится в разделе Циклы/Loops)

Цикл позволяет повторять определённые действия несколько раз. Это как играть песню на повторе - одни и те же действия выполняются снова и снова, пока не выполнится условие остановки.

4. Условия (Conditions) (находится в разделе Логика/Logic)

Условные конструкции позволяют программе принимать решения. Как светофор: если горит красный - стоим, если зелёный - идём. Программа проверяет условие и выполняет соответствующий блок кода.

Практическое задание

Задание 1: Вывод имени

Цель: Создать программу, которая сохраняет имя и показывает его на экране.

Шаги выполнения:

  1. Создайте ящик (переменную) для хранения имени
  2. Положите в этот ящик какое-нибудь имя
  3. Покажите содержимое ящика на экране

Задание 2: Сложение двух чисел

Цель: Сложить два числа и показать результат.

Шаги выполнения:

  1. Создайте два ящика для хранения чисел
  2. Положите в каждый ящик по числу
  3. Сложите содержимое ящиков
  4. Покажите результат сложения

Важные понятия

  • Переменная - именованное место в памяти для хранения данных
  • Вывод - способ показать информацию пользователю
  • Параметр - значение, которое принимает функция
  • Возвращаемое значение - результат работы функции
  • Цикл - конструкция для повторения действий
  • Условие - конструкция для принятия решений в программе
