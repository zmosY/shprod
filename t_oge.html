Задание 1: Количественные параметры информационных объектов

Общий алгоритм решения:

1. Подсчитайте количество различных значений (N) в заданном диапазоне

2. Используйте формулу: 2ⁿ ≥ N, где n - искомое количество бит

3. Найдите минимальное значение n, удовлетворяющее неравенству

4. Умножьте количество бит на длину последовательности

5. При необходимости переведите биты в байты (разделите на 8)

Пример задания:

В системе используется 7-битная кодировка ASCII. Определите объем памяти в байтах для хранения 24 символов.

Решение:

1. Один символ занимает 7 бит

2. 24 символа: 7 × 24 = 168 бит

3. Переводим в байты: 168 ÷ 8 = 21 байт

Ответ: 21

Задание 16: Программирование

Общий алгоритм решения:

1. Определите условия задачи

2. Выделите входные и выходные данные

3. Разработайте алгоритм для решения

4. Реализуйте код

5. Проверьте результат на тестовых данных

Пример задания:

Напишите программу, которая сортирует массив чисел.

Алгоритмическое решение (сортировка пузырьком):

1. Сравниваем соседние элементы

2. Если левый больше правого - меняем местами

3. Повторяем, пока массив не будет отсортирован

Программное решение (Python):

def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr

numbers = [64, 34, 25, 12, 22, 11, 90]
sorted_numbers = bubble_sort(numbers.copy())
print(sorted_numbers)  # Вывод: [11, 12, 22, 25, 34, 64, 90]

Ответ: [11, 12, 22, 25, 34, 64, 90]
